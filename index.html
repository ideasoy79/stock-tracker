주식 입문자

왜 사고 팔았는지 기록하는 습관부터. 기록이 쌓이면 실수 패턴이 보여요.

스윙 트레이더

1~5일 보유 전략을 체계적으로 기록하고 승률과 손익비를 트래킹해요.

학습 중심 투자자

수익보다 판단의 질을 높이는 데 집중. Claude AI와 매일 브리핑 받기.

🚀 사용 방법
1

아침 — 데일리 브리핑

☀️ 브리핑 버튼 → 내 종목 현황이 자동 정리돼서 Claude로 전달.

☀️ 버튼 → Claude 대화 → 오늘 분석 + 스윙 후보 확인
2

장중 — 판단 즉시 기록

매수했든 패스했든 모든 판단을 "오늘" 탭 빠른 기록에 남겨요.

예: "SK하이닉스 / 관망 / 나스닥 선물 -2%, 진입 보류"
3

종목 탭 — 상세 기록

매수 종목은 + 버튼으로 상세 기록. 수익률·손익 자동 계산.

진입가 × 수량 = 투자금 / (매도가-진입가) × 수량 = 손익
4

저녁 — 복기 & 패턴

패턴 탭에서 승률·손익비·실수 패턴 확인. 5건 이상부터 의미 있어요.

갭추격 승률 20% vs 눌림목 승률 70% → 전략 조정
5

주 1회 — 데이터 백업

💾 버튼으로 JSON 저장. 기기 교체 시 📂 복원으로 불러오기.

파일명: tracker-backup-2026-06-18.json
❓ 자주 묻는 질문
데이터는 어디에 저장되나요?
가입한 메일 주소를 통해 원격 Firebase 클라우드 DB(Firestore)에 안전하게 실시간 저장됩니다. 서버망에 안전하게 분산 업로드되므로 다른 모바일 기기, 아이패드, PC 브라우저 등에서 같은 메일로 연동하면 실시간으로 데이터를 이어받을 수 있습니다. 추가로 로컬 수동 보존을 위해 주 1회 수동 💾 백업을 권장해 드립니다.
Claude AI 기능은 어떻게 작동하나요?
☀️ 브리핑 / 🤖 복사 버튼을 누르면 내 트래커 데이터가 자동 정리돼서 클립보드에 특수 텍스트 형태로 동적 임베딩됩니다. 복사된 상태로 곧바로 Claude 연동창으로 연동되며, 대화 입력창에 바로 붙여넣기(Ctrl+V) 하여 즉시 투자 전략 분석 피드백을 수신할 수 있습니다.
종목이 검색 안 돼요
국내 주요 상위 우량 종목이 시스템 DB에 내장되어 있습니다. 최근 상장한 소형주나 가상화폐, 혹은 해외 주식을 기록하시려는 경우 그냥 검색창에 "테슬라" 혹은 "원하는 명칭"을 입력하고 엔터를 치시거나 '직접 종목 등록' 드롭다운 버튼을 눌러 자유롭게 기록 및 관리가 가능합니다.
패턴 분석은 언제 볼 수 있나요?
익절/손절 완료 기록이 최소 1건 이상 있으면 분석 통계 대시보드가 오픈됩니다. 통계적 편차를 줄이고 신뢰도 높은 강약점 분석 리포트를 보시려면 최소 5건 이상의 완료 저널을 누적하는 것을 권해 드립니다.
다른 기기에서 쓸 수 있나요?
동일한 이메일 계정으로 로그인해 주시면 원격 클라우드를 통해 다른 기기에서도 별도의 수동 복사 없이 실시간 연동하여 데이터를 사용할 수 있습니다. 임시 오프라인 상황이라면 💾 백업으로 JSON 저장 후, 다른 기기에서 📂 복원으로 수동 마이그레이션도 지원합니다.
