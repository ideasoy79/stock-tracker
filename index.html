주식 입문자

왜 사고 팔았는지 기록하는 습관부터. 기록이 쌓이면 실수 패턴이 보여요.

📊

스윙 트레이더

1~5일 보유 전략을 체계적으로 기록하고 승률과 손익비를 트래킹해요.

🧠

학습 중심 투자자

수익보다 판단의 질을 높이는 데 집중. Claude AI와 매일 브리핑 받기.

🚀 사용 방법
1

아침 — 데일리 브리핑

☀️ 브리핑 버튼 → 내 종목 현황이 자동 정리돼서 Claude로 전달.

☀️ 버튼 → Claude 대화 → 오늘 분석 + 스윙 후보 확인
2

장중 — 판단 즉시 기록

매수했든 패스했든 모든 판단을 "오늘" 탭 빠른 기록에 남겨요.

예: "SK하이닉스 / 관망 / 나스닥 선물 -2%, 진입 보류"
3

종목 탭 — 상세 기록

매수 종목은 + 버튼으로 상세 기록. 수익률·손익 자동 계산.

진입가 × 수량 = 투자금 / (매도가-진입가) × 수량 = 손익
4

저녁 — 복기 & 패턴

패턴 탭에서 승률·손익비·실수 패턴 확인. 5건 이상부터 의미 있어요.

갭추격 승률 20% vs 눌림목 승률 70% → 전략 조정
5

주 1회 — 데이터 백업

💾 버튼으로 JSON 저장. 기기 교체 시 📂 복원으로 불러오기.

파일명: tracker-backup-2026-06-18.json
❓ 자주 묻는 질문
데이터는 어디에 저장되나요? +
내 기기 브라우저(localStorage)에만 저장돼요. 서버에 전송되지 않아요. 브라우저 캐시 삭제 시 데이터가 사라질 수 있으니 주 1회 백업 권장.
Claude AI 기능은 어떻게 작동하나요? +
☀️ 브리핑 / 🤖 복사 버튼을 누르면 내 트래커 데이터가 자동 정리돼서 Claude 대화창으로 연결돼요. 별도 설정 없이 claude.ai에서 바로 사용 가능.
종목이 검색 안 돼요 +
약 2,756개 종목이 등록돼 있어요. 최근 상장 소형주는 없을 수 있어요. 이 경우 "종목명 코드" 형식으로 직접 입력하면 돼요.
패턴 분석은 언제 볼 수 있나요? +
익절/손절 완료 기록이 1건 이상 있으면 시작돼요. 5건 이상부터 의미 있는 패턴이 보여요.
다른 기기에서 쓸 수 있나요? +
💾 백업으로 JSON 저장 후, 다른 기기에서 📂 복원으로 불러오면 돼요. 현재 자동 동기화는 지원하지 않아요.
