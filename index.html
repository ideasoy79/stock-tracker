학습 중심 투자자

수익보다 판단의 질을 높이는 데 집중. Gemini AI와 매일 브리핑 받기.

🚀 사용 방법
1

아침 — 데일리 브리핑

AI 브리핑 버튼 → 내 종목 현황이 자동 분석되어 시장 전략 수신.

☀️ 브리핑 → 오늘 분석 + 스윙 후보 확인
2

장중 — 판단 즉시 기록

매수했든 패스했든 모든 판단을 "홈" 탭 빠른 기록에 남겨요.

예: "SK하이닉스 / 관망 / 나스닥 선물 -2%, 진입 보류"
3

종목 탭 — 상세 기록

매수 종목은 + 버튼으로 상세 기록. 수익률·손익 자동 계산.

진입가 × 수량 = 투자금 / (매도가-진입가) × 수량 = 손익
4

저녁 — 복기 & 패턴

패턴 탭에서 승률·손익비·실수 패턴 확인. 3건 이상부터 의미 있어요.

갭추격 승률 20% vs 눌림목 승률 70% → 전략 조정
5

주 1회 — 데이터 백업

백업 버튼으로 JSON 저장. 기기 교체 시 복원으로 불러오기.

파일명: tracker-backup-2026-06-18.json
❓ 자주 묻는 질문
데이터는 어디에 저장되나요?
내 기기 브라우저(localStorage)와 원격 클라우드 DB에 동시 저장되어 이메일 로그인 시 다른 기기에서도 실시간 연동됩니다. 데이터 유실 대비를 위해 주 1회 로컬 백업도 권장합니다.
AI 브리핑 기능은 어떻게 작동하나요?
상단 'AI 브리핑' 버튼을 누르면 내 트래커 데이터가 자동으로 요약되어 앱 내부에 팝업 창을 띄웁니다. 앱을 벗어나지 않고도 즉각적인 조언과 스윙 후보를 추천받고 바로 세팅할 수 있습니다.
종목이 검색 안 돼요
수천 개의 국내 종목이 내장되어 있습니다만, 혹시 해외 주식이나 가상화폐, 신규 상장주를 기록하시려면 검색창에 입력 후 맨 하단에 나타나는 '직접 등록' 버튼을 눌러 자유롭게 추가할 수 있습니다.
패턴 분석은 언제 볼 수 있나요?
익절/손절이 완료된 기록이 최소 3건 이상 있으면 활성화됩니다. 데이터가 쌓일수록 나의 강점 태그와 약점 태그가 정밀하게 분석됩니다.
